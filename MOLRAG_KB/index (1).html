MolRAG Architecture Workflow

Training-Free Molecular Property Prediction with Large Language Models

Input

The process begins with a query molecule whose property needs to be predicted (e.g., "Is this molecule toxic?")

Example: SMILES string like "CC(C)Cc1ccc(cc1)C(C)C(O)=O" representing molecular structure
1
Molecular Retrieval

Convert the query molecule into a fingerprint and search for structurally similar molecules with known properties

Fingerprint Encoding
Morgan Fingerprint converts molecular structure into fixed-length vectors capturing topology and connectivity patterns
Similarity Search
Uses Dice coefficient to measure feature overlap and retrieve top-k most similar molecules from reference database
Retrieved Examples
Returns similar molecules with their known properties and similarity scores (e.g., 0.85 similarity, Property: Toxic)
2
Chain-of-Thought Reasoning

Apply domain-specific reasoning strategies to guide the LLM's analysis of molecular properties

Structure-Aware CoT
Systematic structural decomposition: extract functional groups, count aromatic rings, analyze chirality and chain length, then map to properties
Similarity-Aware CoT
Leverage property continuity principle: structurally similar molecules have similar properties. Perform cross-molecule correlation analysis
Performance: Sim-CoT outperforms Struct-CoT on 6 of 7 datasets, suggesting similarity-based reasoning better captures chemical property landscapes
3
Prompt Engineering & LLM Inference

Construct a comprehensive prompt integrating all components and let the LLM generate step-by-step reasoning

Prompt Components
Background info, task instructions, retrieved molecules with similarity scores, CoT strategy steps, strict output format constraints
LLM Processing
Model processes rich context to generate interpretable reasoning trace before final classification (Yes/No)
In-Context Learning
4-shot configuration provides optimal performance within context window constraints (1-shot < 2-shot < 4-shot)

Output

The system produces both a final prediction and human-readable reasoning that chemists can validate

Prediction
Binary classification: "Yes" or "No" for the queried molecular property
Reasoning Trace
Interpretable step-by-step explanation grounded in chemical principles and retrieved examples
Confidence Signal
Similarity scores and reasoning quality provide implicit confidence indicators for validation
